# 🎬 ConsulTube
2025-2학기 빅데이터 프로젝트

<br>

# 📌 프로젝트 개요

**ConsulTube**는 초보 유튜브 크리에이터가 콘텐츠 전략을 효과적으로 수립할 수 있도록  
최근 트렌드, 제목 패턴, 해시태그 효과를 데이터 기반으로 분석·추천하는 플랫폼입니다.

단순 조회수 나열이 아닌,  
“어떤 주제 / 어떤 제목 / 어떤 해시태그”가 효과적인지  
인사이트를 제공하여 시행착오를 줄이고 성장 속도를 높입니다.

<br>

# 📊 주요 기능

### 1) 콘텐츠 트렌드 분석
- YouTube Data API 기반 데이터 수집
- 최근 인기 주제 토픽 모델링 (예: BERTopic)
- 조회수/빈도 기반 트렌드 TOP N 추천


### 2) 해시태그 효과 분석
- 태그별 조회수·좋아요 지표 분석
- 상관분석/회귀분석 기반 성과 측정
- **“추천 태그 조합”** 제시


### 3) 제목 텍스트 분석 & 추천
- 인기 영상 제목의 문장 구조 패턴 분석
- NLP 기반 키워드·감성 분석
- 활용 가능한 제목 템플릿 추천


### 4) 카테고리 기반 선택 구조
- **[라이프]** 뷰티 / 패션 / 먹방 / 일상  
- **[취미]** 게임 / 음악 / 운동 / DIY  
- **[여행·문화]** 여행 / VLOG / 공연 / 장소리뷰  
- **[지식·정보]** 교육 / 금융 / 커리어 / 자기계발  
- **[리뷰]** IT제품 / 화장품 / 생활 리뷰  
- **[엔터]** 뉴스 / 이슈 / K-pop / 챌린지
